# 🙋‍♀️프로젝트 명 ‘출첵’

  ###### 출첵은 직원들의 스케줄과 출근부를 한 눈에 보여주고, 근무 통계를 제공하는 근무 관리 서비스 입니다.
  
  ###### 매장 내 포스기와 근무지 밖에서도 사용할 수 있는 웹 서비스 입니다.
    
    
  # 🗓기능  
  
  ###### - 스케줄 / 출근부 / 메모 통합 캘린더
  ###### - 스케줄과 출근부 기간별 / 직원별 검색
  ###### - 근무 형태 / 직원 관리
  ###### - 출근부 엑셀 파일 다운로드
  ###### - 출근 시간 통계 제공
